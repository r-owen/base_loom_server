source dataclass CurrentEndNumber(total_end_number0: int, total_end_number1: int, end_number0: int, end_number1: int, end_repeat_number: int)

The current threading end numbers and repeat number

The range of end numbers is [end_number0, end_number1). If end_number0 is 0 then end_number1 will also be zero (no threads in range).

source dataclass ShaftState(state: ShaftStateEnum, shaft_word: int)

Shaft status

shaft_word is a bitmask:

  • bit 0 = shaft 1, etc.
  • bit value is 0 if shaft is up

shaft_word is only meaningful if state = ShaftStateEnum.DONE
