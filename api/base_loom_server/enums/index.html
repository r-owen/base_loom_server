source enum DirectionControlEnum(*args, **kwds)

Bases : enum.IntEnum

What controls the direction (weave/unweave).

  • FULL: The direction can be changed by both the web interface and the direction button on the loom. If the loom supports this (e.g. Séguin), it is the only allowed value. If the loom does not (e.g. Toika), this value is prohibited.
  • LOOM: The direction can only be changed by the unweave button on the loom. The web browser only displays the direction.
  • SOFTWARE: The direction can only be changed by the web interface. The unweave button on the loom is ignored.
