source async method BaseMockLoom.oob_command(self, cmd: str) → None

Handle out-of-band commands.

Find a method named "oob_command_{cmdchar}", where cmdchar = cmd[0], and call it with one positional argument: cmd.

None of the standard oob commands pay attention to the cmd argument, but subclasses may add other oob commands, and those may accept data.
