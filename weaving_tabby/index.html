Weaving Tabby

The Tabby mode allows you to weave tabby (plain weave), or a close approximation. This can be used to weave hems or item separators.

Caveats:

  • The software does the best it can to determine suitable tabby picks, but the algorithm is not perfect. If you are unsatisified with the tabby pattern, create your own tabby pattern file, upload it, and weave it using the Weaving panel.

  • This is not intended to intersperse tie down picks with pattern picks in patterns such as overshot. Tie-down picks should be part of the pattern file.

The two tabby picks are always opposites: if a shaft is raised in one pick, it will be lowered in the other. However, shafts beyond the last threaded shaft are always lowered.

The following assumes you have done all the basics:

  • Connected your web browser to the loom server
  • Uploaded at least one pattern, and selected a pattern from the pattern menu
  • Selected the Tabby mode.

Pattern Display

The tabby pattern is displayed as a picture that shows woven tabby fabric in the bottom half, and potential future fabric, somewhat grayed out, in the top half.

A setting allows you to specify whether warp end 1 is shown at the right or left.

Note that the display is naive: it shows all threads as the same thickness.

Weave Direction

The "Direction" button shows "Weave" or "Unweave".

To change between threading and unthreading see Weaving: Direction.

Note that there is rarely any reason to unweave tabby: backwards is the same as forwards, except you cannot back up beyond pick 0.

Jumping

You can jump to a different pick. See Weaving: Jumping for details.

Note that there is rarely any reason to jump when weaving tabby.
